# 2. 알고리즘 (의사 코드) (10점)

## 1. `main` 함수의 전체 흐름
시작
반복 (사용자가 '종료'를 선택할 때까지):
    메뉴 출력 (1. 도서 추가, 2. 도서 삭제, 3. 도서 검색, 4. 도서 수정, 5. 도서 목록 출력, 0. 종료)
    사용자 선택 입력
    만약 선택이 1이면:
        도서 추가 함수 호출
    만약 선택이 2이면:
        도서 삭제 함수 호출
    만약 선택이 3이면:
        도서 검색 함수 호출
    만약 선택이 4이면:
        도서 수정 함수 호출
    만약 선택이 5이면:
        도서 목록 출력 함수 호출
    만약 선택이 0이면:
        반복 종료
반복 끝
종료


## 2. 핵심 기능 1: (도서 추가 함수)
시작
    만약 현재 도서 수가 최대 도서 수 (예: 5권)를 초과하면:
        "도서 추가 불가" 메시지 출력
        함수 종료
    아니면:
        도서 이름 입력받기
        저자 이름 입력받기
        출판년도 입력받기
        장르 입력받기
        입력받은 도서 정보를 배열에 저장
        현재 도서 수 증가
    함수 종료
종료


## 3. 핵심 기능 2: (도서 삭제 함수)
시작
    삭제할 도서 번호 입력받기
    만약 입력된 번호가 유효하지 않으면 (즉, 1부터 현재 도서 수까지의 범위 밖이면):
        "잘못된 번호입니다" 메시지 출력
        함수 종료
    아니면:
        삭제할 도서 번호에 해당하는 도서 정보 삭제
        삭제 후, 뒤에 있는 도서들 한 칸씩 앞으로 이동
        현재 도서 수 감소
    함수 종료
종료
## 4. 핵심 기능 3: (도서 검색 함수)
시작
    검색어 입력받기 (도서 이름 또는 저자 이름)
    검색 결과가 있을 때까지 반복:
        도서 목록에서 하나씩 비교:
            만약 도서 이름 또는 저자 이름이 검색어와 일치하면:
                해당 도서 정보 출력
        반복 종료
    만약 검색 결과가 없으면:
        "검색 결과가 없습니다" 메시지 출력
    함수 종료
종료
## 5. 핵심 기능 4: (도서 수정 함수)
시작
    수정할 도서 번호 입력받기
    만약 입력된 번호가 유효하지 않으면 (즉, 1부터 현재 도서 수까지의 범위 밖이면):
        "잘못된 번호입니다" 메시지 출력
        함수 종료
    아니면:
        수정할 도서의 정보를 입력받기:
            도서 이름, 저자 이름, 출판년도, 장르 입력받기
        입력받은 값을 해당 도서의 정보에 저장
    함수 종료
종료
## 6. 핵심 기능 5: (도서 목록 출력 함수)
시작
    도서 목록이 비어있는지 확인:
        만약 비어 있으면:
            "도서 목록이 비어 있습니다" 메시지 출력
            함수 종료
    아니면:
        도서 목록 출력 (도서 번호, 이름, 저자, 출판년도, 장르)
    함수 종료
종료
